# Установка 
```shell
npm config set @ippu:registry=https://git.ds.ecpk.sibintek.ru/api/v4/groups/500/-/packages/npm/
```

```shell
npm config set -- //git.ds.ecpk.sibintek.ru/:_authToken=<ТОКЕН ДОСТУПА ПОЛЬЗОВАТЕЛЯ>
```

```shell
npm set strict-ssl false
```

```shell
npm i @ippu/ippu-front-foundation
```
